Getting Started with
Convertigo Mobility Platform

The Convertigo platform is the leader open source low-code MXDP (Multi eXperience Development Platform) composed of the MBaaS server (Mobile back end as a Service) providing mobile micro-services, back end storage, Data sync and the Mobile Builder Visual Low-Code app builder able to build and run Hybrid Cordova Angular / Ionic apps using a visual approach.


In order to get you up and running, we prepared a set of “getting started” videos you can use to develop your first enterprise mobile application projects.
